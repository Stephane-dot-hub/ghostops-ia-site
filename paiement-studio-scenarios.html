GhostOps IA – Niveau 2 · Studio Scénarios

Paiement sécurisé – GhostOps Studio Scénarios

Vous êtes sur le point d’activer le Niveau 2 de l’offre GhostOps IA : GhostOps Studio Scénarios.

Ce niveau a pour objectif de transformer votre situation – et le cas échéant les éléments issus d’un Diagnostic IA GhostOps – en 2 à 3 scénarios tactiques détaillés, avec matrices risques / gains / exposition, afin de préparer des arbitrages assumables (direction générale, DRH, board, actionnaires).

L’honoraire de référence pour GhostOps Studio Scénarios est fixé à 1 490 € TTC, pour un dossier standard (une entité, un périmètre délimité, un board ou comité d’audit identifié).

Le règlement est opéré via Stripe (paiement sécurisé). Les éléments précis (montant, devise, récapitulatif) sont affichés sur la page Stripe avant validation. Aucun débit n’est effectué tant que vous n’avez pas validé le paiement sur cette page.

Paiement annulé.

Aucune somme n’a été débitée. Vous pouvez relancer le paiement ou revenir au Studio Scénarios.

Procéder au paiement sécurisé avec Stripe

Montant du GhostOps Studio Scénarios : 1 490 € TTC

Le règlement s’effectue via Stripe, solution de paiement sécurisée. Vous serez redirigé vers une page de paiement HTTPS, puis renvoyé automatiquement vers GhostOps Studio Scénarios une fois le paiement validé.

Aucune information de carte bancaire n’est stockée sur le site GhostOps : toutes les données de paiement sont traitées par Stripe.

En cas de difficulté technique, vous pouvez nous signaler le problème via la page Contact. Aucun débit n’est effectué tant que vous n’avez pas validé le paiement sur la page Stripe.

Le règlement porte exclusivement sur le Niveau 2 – GhostOps Studio Scénarios. Le Diagnostic IA (niveau 1) et le Pré-brief Board (niveau 3) font l’objet d’honoraires distincts.
